Revisor

Fat-free integration testing

  • GitHub Project
  • Issues
  • User Group

Revisor is a tiny WebKit-based browser which listens for HTTP commands and executes them. It's advantages:

  • C++ / Qt 4 / WebKit

    Fast, memory-efficient and cross-platform.

  • HTTP / JSON

    Can be used from any environment which is able to send HTTP requests.

  • No Fat Browsers

    You don't need to run entire browser with all it's unnecessary components.

  • Ruby Helper Lib

    Revisor ships with a small Ruby library which will make your life easier.

  • KISS / DRY

    Small codebase and easy to learn API.

  • SOMETHING ELSE?

    Could not think up what to put here, but I need 6 blocks for symmetry. Just ignore it.

    • NB: Revisor is under active development. It's unstable, has almost no documentation and possibly contains a lot of bugs.
